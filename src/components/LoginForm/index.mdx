# LoginForm
Componente padrão utilizado para realizar *login* nas aplicações da Ziro. **LoginForm** se trata de um formulário contendo os campos referentes ao *email* e a senha do usuário, onde ambos possuem suas respectivas funções de validação. Um detalhe a ser observado na utilização deste componente é que a propriedade **sendToBackend** necessita que seja feito um encadeamento de funções, utilizando assim uma função intermediária.

<Playground>
  <div style={{height: 320, ...containerWithoutHeight}}>
		<LoginForm
			sendToBackend={state => () => new Promise((res, rej) => setTimeout(() => res('Enviado!'), 1500))}
		/>
	</div>
</Playground>
