# UpdatePass
Componente padrão utilizado no fluxo de alteração de senha. Um detalhe a ser observado na utilização deste componente é que a propriedade **sendToBackend** necessita que seja feito um encadeamento de funções, utilizando assim uma função intermediária.

<Playground>
    <div style={{height: 500, ...containerWithoutHeight}} >
        <UpdatePass sendToBackend={() => () => new Promise((res, rej) => setTimeout(() => res('Enviado!'), 1500))} />
    </div>
</Playground>
