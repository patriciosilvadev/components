# CameraOverlay
Componente exibido por cima da camera, quando a câmera está ligada, mostra os botões para tirar foto, ligar e desligar a câmera, trocar de câmera, assim como um botão de fechar.
Quando a câmera está no modo preview, mostra os botões de excluir e enviar
