# Error

Conjunto de componentes que são utilizados para informar ao usuário quando ocorre algum tipo de erro. Vale ressaltar que esses componentes em situações reais são usados para preencher a tela inteira com o erro, mas nas seguintes demonstrações serão apresentados em tamanhos reduzidos. São eles:

- **Not Found**: Página não encontrada.
- **Boundary**: Erro na renderização da página.
- **Undef**: Ocorreu algum erro.
