# LoginTrouble
Componente padrão utilizado para resolver problemas ocorridos no *login*. **LoginTrouble** apresenta ao usuário uma mensagem e botões contendo as opções disponíveis para efetuar a correção do erro, bem como a opção de voltar inclusa no cabeçalho (componente **Header**).
