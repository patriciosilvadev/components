# HeaderHome
**HeaderHome** é um componente dado pela composição do componente **Icon** (mais específicamente o ícone da Ziro) com um *link* de navegação. Ele é utilizado mais comumente em ocasiões no qual se deseja disponibilizar um *link* direto para a página de cadastro ou de *login*.
