# DeleteAccount
Componente padrão utilizado para a deleção de uma conta. Um detalhe a ser observado na utilização deste componente é que a propriedade **sendToBackend** necessita que seja feito um encadeamento de funções, utilizando assim uma função intermediária.

<Playground>
    <div style={{height: 300, ...containerWithoutHeight}} >
        <DeleteAccount sendToBackend={() => () => new Promise((res, rej) => setTimeout(() => res('Enviado!'), 1500))} />
    </div>
</Playground>
